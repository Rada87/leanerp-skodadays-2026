# LeanERP × Škoda Days 2026

Jednostránková prezentace o třech slidech. Obsah jednotlivých slide sekcí je v `src/public/index.html`; vzhled v `src/public/styles.css` a ovládání v `src/public/script.js`.

## Struktura

```text
docs/         # projektová dokumentace
src/
  public/    # versionované soubory dostupné přímo přes web
  app/       # versionovaný neveřejný aplikační kód
  data/      # lokální neversionovaná data
tmp/          # lokální dočasné soubory
```

Pro lokální náhled stačí otevřít `src/public/index.html` v prohlížeči.

Nasazení je popsáno v [deploy.md](deploy.md).
